## Overview
Welcome to my R programming portfolio. This repository serves as a comprehensive documentation of my journey through **Data Science**, **Statistical Analysis**, and **Machine Learning**. 

It progresses from foundational R syntax to complex predictive modeling, utilizing advanced techniques like **Bagging**, **Decision Trees**, and **Hypothesis Testing**. All projects are documented using **Quarto (.qmd)** for reproducible research.

## Repository Modules

| Module | Description | Key Libraries |
| :--- | :--- | :--- |
| **[1. Introduction to R](./Introduction%20to%20R)** | Core syntax, vector operations, and data cleaning with `dplyr`. | `base`, `readr`, `dplyr` |
| **[2. EDA](./EDA)** | Exploratory Data Analysis, data visualization, and correlation studies. | `ggplot2`, `MASS`, `psych`, `plyr` |
| **[3. Models](./Models)** | Implementation of ML algorithms (KNN, Trees, Regression) and Statistical Tests. | `caret`, `rpart`, `ipred`, `leaps`, `ISLR` |

## Tools & Technologies
The following libraries and tools are heavily used throughout this project:

* **Data Manipulation:** `dplyr`, `tidyr`, `stringr`, `plyr`
* **Visualization:** `ggplot2`, `lattice`, `rattle`, `rpart.plot`
* **Machine Learning:** `caret`, `rpart`, `class` (KNN), `e1071`, `ipred` (Bagging)
* **Statistics:** `MASS`, `stats` (Hypothesis testing), `leaps` (Subset selection)
* **Reporting:** `knitr`, `flextable`, `broom`
